# USB Manager - Feature Ideas

## High Impact

- [x] **Duplicate detection** - Check if file exists at destination before copying (skip/overwrite/rename options)
- [x] **Safe eject** - Button to safely unmount USB after copy completes
- [ ] **Date-based organization** - Auto-organize into `~/Photos/2025/01/` folders based on file date
- [x] **Exclusion patterns** - Global ignore list for `.DS_Store`, `Thumbs.db`, `__MACOSX`, etc.

## Nice to Have

- [ ] **Copy history** - Log of what was copied, when, from which USB
- [x] **Search/filter** - Filter the file tree by name or extension
- [x] **File type icons** - Visual distinction for images, videos, documents
- [ ] **Desktop notifications** - Alert when USB inserted or copy complete
- [ ] **Verification** - Checksum verification after copy to ensure integrity

## Power User

- [ ] **USB profiles** - Different rule sets per USB (work vs personal drive)
- [ ] **Batch rename** - Add prefix/date/sequence during copy
- [x] **Selective auto-copy** - Uncheck specific files in auto mode before confirming
